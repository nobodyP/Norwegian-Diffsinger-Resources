# Phoneme Info

> **phoneme system based on Arpabet, German (Marzipan,) and polish**

### Non-vocal phonemes

|   Use        | Phoneme | Description |
| :----------: | :-----: | :---------: |
| Silence      |    SP   | pure silence without sound |
| Breath       |    AP   | breaths |
| Vocal fry    |    vf   | - |
| Glottal stop |    cl   | abrupt stops |
| Exhale       |   exh   | unvoiced exhale (no pitch)|
|Voiced exhale |   axh   | voiced exhale (has pitch)|
